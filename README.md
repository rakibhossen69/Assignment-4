#  What is Event Bubbling? And how does it work =
Event bubbling হল JavaScript-এ DOM-এর মাধ্যমে event child element থেকে parent element-এর দিকে উঠে। আমি যদি event.parent করি, তাহলে যে element-এ ক্লিক করেছি তার parent element-পাব, এবং আমি যদি event.children করি, তাহলে যে element-এ ক্লিক করেছি তার child element-গুলো পাব।
# What is the difference between preventDefault() and stopPropagation() methods= 
preventDefault() এবং stopPropagation() হল দুইটি event method। preventDefault() default behavior রোধ করে, আর stopPropagation() event propagation বন্ধ করে।
